# MCP ガイドライン

Model Context Protocol (MCP) ツールの開発方針に関するドキュメントです。  
ここでは「Ateliers プロジェクトで MCP をこう使っています」というスタイルが反映されています。

## 概要

このディレクトリには、MCP ツール開発における以下のガイドラインが含まれます：

- **ツール設計**: Description 属性の記述方法、命名規則、設計パターン
- **実装ベストプラクティス**: コーディング規約、テスト方針
- **統合ガイド**: サービス層との連携、エラーハンドリング

## 対象者

- MCP ツールを新規作成する開発者
- 既存の MCP ツールを改修する開発者
- MCP ツールの Description を最適化したい開発者
- LLM との連携を改善したい開発者
